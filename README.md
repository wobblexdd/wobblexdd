## Featured Projects

### WobbleAuction
Modern GUI-based auction house plugin for Paper 1.21+ with pagination, search, expiration handling, SQLite storage, and economy integration.

### WobbleBounty
Dynamic bounty system with GUI menus, player targeting, automated payouts, and competitive SMP-oriented gameplay flow.

### WobbleCombat
Combat control and anti-logout plugin designed to reduce abuse and keep PvP fair.

### WobbleTeams
Team management system with invitations, roles, persistence, and scoreboard integration.

### WobbleSettings
Centralized player settings system designed to control and synchronize behavior across multiple plugins.

### WobbleKillSound
Configurable PvP kill sound plugin with custom feedback handling.

### WobbleRTP
High-performance random teleport plugin with GUI-based world selection, safe location search, and async-oriented teleport flow.

### MoroSMP Platform
Full-stack MERN platform connected to Minecraft server data, community systems, and internal backend services.

### Automation Suite
Private Python toolkit for automation, project cleanup, patching, workflow optimization, and server-side utility tasks.
